## 二、关键发现

### 发现1：产品类型图片（重要！）

**参考网站有**：
- "Types of Aluminum LED Profiles" 图片展示
- 展示同一类型产品的不同变体
- 例如：Surface mounted, Embedded, Corner, Wall-mounted等

**当前系统缺少**：
- 没有产品类型图片字段
- 模板中也没有产品类型图片

**建议**：
- 在模板中添加"产品类型图片"字段
- 可以是多张图片，展示同一类型产品的不同形式

---

### 发现2：产品特性图片（重要！）

**参考网站有**：
- "feature of aluminum profiles" 图片
- 展示产品的特性/优势

**当前系统缺少**：
- 没有特性图片字段

**建议**：
- 在模板中添加"特性图片"字段
- 可以是一张或多张图片

---

### 发现3：化学组成表格（通用内容）

**参考网站有**：
- 完整的化学组成表格
- 包含多种合金的化学元素含量

**当前系统缺少**：
- 没有化学组成数据

**建议**：
- 在模板中添加"化学组成表格"数据
- 可以是一个JSON结构或关联模型

---

### 发现4：顶部信息栏字段差异

**参考网站字段**：
- Application, Shape, Color, Material & Temper, Thickness, Precision, Surface Treatment, Machining, Deep Process

**当前系统字段**：
- range_param, type_param, surface_treatment, colors, grade, temper, + 商业信息

**差异**：
- 参考网站更注重技术参数（Shape, Precision, Machining, Deep Process）
- 当前系统更注重商业信息（OEM, Samples, Supply Ability等）

**建议**：
- 可以同时保留，但需要调整展示布局
- 顶部展示技术参数（参考网站风格）
- 商业信息可以放在参数区域或单独区域

## 三、模板系统需要完善的内容

### 1. 产品类型图片（Product Type Images）

**需求**：
- 展示同一类型产品的不同变体
- 例如：Surface mounted, Embedded, Corner等LED型材类型

**实现方案**：
```python
# 在ProductTemplate模型中添加
class TemplateProductTypeImage(models.Model):
    """模板产品类型图片"""
    template = models.ForeignKey(ProductTemplate, ...)
    title = models.CharField('类型名称', max_length=200)
    image = models.ImageField('类型图片', upload_to='template_product_types/')
    description = models.TextField('类型描述', blank=True)
    order = models.IntegerField('排序', default=0)
```

**前端展示**：
- 在Product Description区域下方
- 网格布局展示产品类型图片
- 每个类型有图片、标题、描述

---

### 2. 产品特性图片（Feature Images）

**需求**：
- 展示产品特性/优势的图片

# 在ProductTemplate模型中添加
class TemplateFeatureImage(models.Model):

    image = models.ImageField('特性图片', upload_to='template_features/')

    order = models.IntegerField('排序', default=0)

**前端展示**：

---

### 3. 化学组成表格（Chemical Composition Table）

**需求**：
- 展示不同合金的化学组成
- 表格格式：合金名称 | 各元素含量

**实现方案**：
```python
# 在ProductTemplate模型中添加
class TemplateChemicalComposition(models.Model):
    """模板化学组成"""
    template = models.ForeignKey(ProductTemplate, ...)
    alloy_name = models.CharField('合金名称', max_length=50)  # 如：6063
    si = models.CharField('Si含量', max_length=50, blank=True)
    fe = models.CharField('Fe含量', max_length=50, blank=True)
    cu = models.CharField('Cu含量', max_length=50, blank=True)
    # ... 其他元素
    order = models.IntegerField('排序', default=0)
```

**或者使用JSON字段**：
```python
# 在ProductTemplate模型中添加
chemical_composition = models.JSONField('化学组成表格', blank=True, null=True)

# JSON格式：
{
  "alloys": [
    {
      "alloy": "6063",
      "si": "0.2-0.6",
      "fe": "<0.35",
      "cu": "<0.1",
      ...
    },
    ...
  ]
}
```

**前端展示**：
- 表格格式
- 第一行是表头（Alloy | Si | Fe | Cu | ...）
- 每行是一个合金的化学组成
